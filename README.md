# Impact of Social Media On Hate Crimes Replication


This project was developed as part of the **Economics of Migration** course at École Polytechnique. It aims at replicating and extending the analysis publised in the paper _"From hashtag to hate crime: Twitter and antiminority sentiment."_ by **Müller, Karsten, and Schwarz (2023)**.
